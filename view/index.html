Quote API

To get a single random quote, request: "https://dw-quote-api.herokuapp.com/quotes"

To get quotes from a particular character, enter their name: "https://dw-quote-api.herokuapp.com/quotes/<name>"

For multiple quotes, define the amount you want returned: "https://dw-quote-api.herokuapp.com/quotes?amount=<number>"

The data will return in JSON format. Here is an example:

        [
          {
          "quote": "I am definitely a madman with a box!",
          "episode": "Season 5, Episode 1",
          "character": "The Doctor"
          }
        ]
