Quote API

To request a single random quote: "https://url.com/quotes"

To request quotes by character: "https://url.com/quotes/<name>"

To request multiple quotes define an amount: "https://url.com/quotes?amount=<number>"

The data will return in JSON format like this:

        [
          {
          "quote": "I am definitely a madman with a box!",
          "episode": "Season 5, Episode 1",
          "character": "The Doctor"
          }
        ]
